    class HTTP::Request

    • HTTP::Request
    • Reference
    • Object

    Overview

    An HTTP request.

    It serves both to perform requests by an HTTP::Client and to represent requests received by an HTTP::Server.

    A request always holds an IO as a body. When creating a request with a String or Bytes its body will be a IO::Memory wrapping these, and the Content-Length header will be set appropriately.

    def path_params : Hash(String, String)? #

    A hash containing path params (extracted from the request's path). It's automatically set when routing with Prism::Router.

    For example, request with path "/user/42/edit" routed with put "/user/:id/edit" will have {"id" => "42"} path params.
